Introdução O que pode ser encontrado neste pacote e onde ele pode ser usado

Obrigado por comprar o template Joomla. Essa documentação consiste em várias partes e lhe mostra o processo inteiro de como configurar e administrar o site em Joomla do zero. Nós demos o nosso melhor para fazer esse manual o mais claro e fácil de se seguir possível.

O que é Joomla CMS?

Joomla CMS é uma solução open source para criação de conteúdo rico para websites. Ele lhe permite construir sites e aplicativos online poderosos e há praticamente nenhuma necessidade de possuir habilidades técnicas ou conhecimento para gerenciar. Em muitos aspectos, incluindo sua facilidade de manuseio e extensibilidade, têm feito o Joomla ser o software para website mais popular disponível. Leia Mais

O que é template Joomla

Template Joomla é um skin para a sua plataforma CMS do Joomla. Em outras palavras, você pode facilmente alterar a aparência do seu site Joomla instalando um novo template em poucas e fáceis etapas. Com toda sua simplicidade, o template Joomla é fornecido com todos os arquivos fontes necessários e você pode editá-lo ou extendê-lo do jeito que você precisar.

Estrutura dos arquivos

O pacote do template que você baixou consiste em várias pastas. Vamos ver o que cada pasta contém:

  • Documentation - contém arquivos de documentação
    • documentation.html - arquivo de documentação principal. Você está bem aqui:)
  • Joomla - contém arquivo do tema Joomla, pacote de instalação rápida e arquivo script descompactado.
    • theme###.zip - arquivo do tema. Contém todos os arquivos do tema. Deve ser instalado através do administrador de extensão Joomla
    • fullpackage.zip - contém pacote de instalação completa do Joomla com arquivos do mecanismo, tema e conteúdo de amostra (reserva de banco de dados, imagens de amostra)
    • unzip.php - arquivo usado para extrair arquivos do fullpackage.zip ao enviar para o servidor de hospedagem.
  • Screenshots - contém prints do template.
  • Sources - contém arquivos de fonte do template
    • psd - arquivos fonte de template do Adobe Photoshop (.psd)
    • to_the_root_folder - contém arquivos de conteúdo de amostra usados para fazer seu site aparecer como uma demonstração de template ao vivo. O conteúdo dessa pasta será enviado ao diretório raiz do seu Joomla.
    • dump.sql - arquivo de despejo de banco de dados. Contém conteúdo de amostra de tema.
    • pkg_bundle.zip - contém extensões adicionais fornecidas com o tema
  • fonts_info.txt - contém links onde as fontes padrões do template podem ser baixadas
  • info.txt - contém instruções de como extrair os arquivos fonte

Preparar Do que você precisa para fazer isso funcionar

Antes que você comece a configurar o seu site Joomla, por favor, certifique-se de que você estará totalmente preparado. Por favor, complete as seguintes etapas de preparação:

Software de edição

Para se sentir confortável ao mexer no template Joomla, nós recomendamos que você baixe todos os aplicativos necessários. Você pode ver a lista do software necessário na página de amostra do template ao vivo.
Os requerimentos podem variar para diferentes templates, então nós lhe avisaremos aqui o que é necessário no geral:

  1. Primeiro de tudo você precisa dos aplicativos corretos para extrair o arquivo protegido por senha sources_#########.zip. Você pode usar o WinZip 9+ (Windows) e o Stuffit Expander 10+ (Mac).
  2. Você pode usar também o Adobe Photoshop. Ele é utilizado para editar arquivos com a extensão .PSD e é necessário se você quiser editar os gráficos e imagens do template.
  3. Para editar os arquivos de código fonte você precisa de algum editor de código como o Adobe Dreamweaver, Notepad++, Sublime Text etc.
  4. Para enviar os arquivos ao servidor de hospedagem você pode precisar de um Gerenciador de FTP como o Total Commander, FileZilla, CuteFTP etc.

Hospedagem

Como o Joomla CMS é um aplicativo com base em PHP/MySQL, você precisa preparar um ambiente de hospedagem para rodar o Joomla.

Se você possuir hospedagem, por favor, certifique-se de que os requerimentos do software Joomla são aceitos e de que está pronto para ser usado com sites Joomla.

Caso contrário, você pode rodar o Joomla localmente no seu computador usando um servidor local. Para criar um servidor de hospedagem local, por favor, utilize os aplicativos da localhost como o WAMP, AppServ, MAMP etc. Qualquer um desses podem ser facilmente instalados como qualquer outro programa e usado para rodar o Joomla.

Por favor, veja os tutoriais abaixo de como configurar o ambiente de desenvolvimento local:

  • Como instalar o ambiente de desenvolvimento web do AppServ
  • Como instalar o ambiente de desenvolvimento web do WAMP
  • Como instalar o ambiente de desenvolvimento web do XAMP

O método de instalação completa pode ser utilizado se você quiser configurar seu site Joomla do zero. Nós supomos que você já concluiu todas as etapas de preparação e tem todas as coisas necessárias.

Criando Banco de Dados

Quando você terminar de enviar os arquivos ao servidor de hospedagem, você pode preparar um novo banco de dados para o seu site Joomla. Você pode criar um novo banco de dados usando uma ferramenta de gerenciamento de banco de dados no painel de controle do seu servidor de hospedagem (normalmente PhpMyAdmin).

Com a ferramenta phpMyAdmin você poderá criar um novo banco de dados em 3 simples etapas: (veja os slides abaixo)

  1. Clique no botão do Banco de Dados (em Inglês, "Database") no menu de nevageção superior para acessar a lista de banco de dados
  2. Nome de entrada do banco de dados (em Inglês, "database") no campo "Criar novo banco de dados" (em Inglês, "Create new database")
  3. Clique em Criar botão (em Inglês, "Create button") no lado direito

Seu banco de dados pode ser visto na lista de banco de dados. Clique no nome dele para acessar o banco de dados

 

  • 1. Clique no botão de Banco de Dados para acessar a lista de banco de dados

  • 2. Nome de entrada do banco de dados
    3. Clique em criar botão (em Inglês, "Create button")

  • 4. Seu banco de dados pode ser visto na lista de banco de dados. Clique no nome para obter acesso ao banco de dados.

 

Você pode também dá uma olhada no tutorial em vídeo de Como criar um banco de dados

Se você tiver quaisquer problemas ao criar um banco de dados, por favor, entre em contato com seu servidor de hospedagem para obter ajuda.

Fazendo upload e descompactando

Para começar a mexer com o Joomla você precisa enviar os arquivos ao seu servidor de hospedagem. Isso pode ser feito usando o gerenciador de sua hospedagem ou algum gerenciador terceirizado de FTP.

  1. Abra seu diretório de pacote de template e vá ao diretório "joomla"
  2. Selecione os arquivos 'unzip.php' e 'fullpackage.zip' e os envie ao seu servidor( Como enviar arquivos ao servidor. )
  3. Digite o caminho para o arquivo 'unzip.php' no seu servidor (http://seu_domínio/unzip.php) no seu navegador.
  4. Você deve ver a seguinte tela:

Img 1. Unzip.php tela inicial.

  1. No campo suspenso 'Escolha seu arquivo zip' (em Inglês, 'Choose your zip file') por favor selecione o arquivo enviado fullpackage.zip.
  2. No campo 'Descompactar para' (em Inglês, 'Unzip to') especifique o diretório pra onde você quer extrair os arquivos.
  3. Clique no botão "Descompactar" (em Inglês, 'Unzip') para confirmar.
Por favor, certifique-se de colocar as permissões corretas para o diretório onde você está planejando em extrair os arquivos. As permissões devem ser CHMOD 755 ou 777, depende da configuração do seu servidor.

Instalação de Dados de Amostra

Os mesmos dados são utilizados para fazer o seu site Joomla aparecer como a página de demonstração ao vivo do template. Ele contém artigos de amostra, imagens, configuração de módulos, configurações de site, etc. Se você não está planejando em usar imagens de amostra ou qualquer outro conteúdo de amostra, por favor, pule esta etapa.

Para instalar os dados de amostra você precisar concluir dois passos simples.

Passo 1. Fazer upload de imagens de amostra.

Para enviar imagens de amostra, abra seu diretório de pacote do template e vá para a pasta "fontes" (em Inglês, "sources"). Lá você deve ver o diretório "to_the_root_directory". Ele contém arquivos que devem ser colocados na raiz de instalação do seu Joomla. Por favor, use o gerenciador de arquivo para enviar arquivos ao seu servidor.

Passo 2. Importar arquivo SQL.

E etapa final e mais importante da instalação de dados de amostra é a importação do arquivo SQL. O arquivo SQL é um despejo do banco de dados de amostra, todos os dados do site em Joomla devem ser armazenados lá. O arquivo SQL pode ser importado usando sua ferramenta de gerenciamento de banco de dados (phpMyAdmin). O arquivo SQL do template - "dump.sql" é localizado no diretório "fontes" (em Inglês, "sources") do pacote do template.

 

Por favor, NÃO importe o arquivo SQL se você tiver um site no ar, pois ele irá substituir o banco de dados do seu site e todos os seus dados serão perdidos.
Cuidado: antes de importar um arquivo SQL, por favor, certifique-se de que o prefixo de sua mesa de banco de dados se coincide com o prefixo da mesa no arquivo SQL.

Você pode verificar o prefixo da mesa diretamente no banco de dados através da ferramenta de gerenciamento do banco de dados. Verifique os nomes das mesas. O prefixo é um conjunto de símbolos antes do nome da mesa (ex.: mesa "jos_assets" o prefixo é "jos_"). Você também pode encontrar o prefixo de banco de dados no arquivo "configuration.php" no diretório raiz do seu Joomla. Procure a linha 19: public $dbprefix = 'jos_'; (onde o prefixo é"jos_")

Se os prefixos da mesa não se coincidem, abra o arquivo "dump.sql" com seu editor de código e usando a ferramenta Pesquise e Substitua (em Inglês, "Search & Replace") (CTRL+F ou CTRL+H) substitua o prefixo da mesa "jos_" (usado no arquivo SQL) com o que coincide com seu banco de dados. Você pode verificar o vídeo tutorial detalhado de como alterar o prefixo do banco de dados no arquivo SQL.

Agora você pode importar o arquivo SQL para o seu banco de dados.
Para importar os dados de amostra ao seu banco de dados, por favor, realize essas simples etapas: (verifique os slides abaixo)

  1. Selecione seu banco de dados da lista de banco de dados na coluna da esquerda na ferramenta phpMyAdmin
  2. Clique no botão "Importar" (em Inglês, "Import") no menu de navegação superior
  3. Clique em "Escolher arquivo" (em Inglês, "Choose file") e procure o arquivo dump.sql no diretório do pacote de templates "fontes (em Inglês, "sources).
  4. Clique no botão "Ir" (em Inglês, "Go") para começar a importar os dados de amostra

 

  • 1. Selecione seu banco de dados

  • 2. Clique no botão "Importar" (em Inglês, "Import")

  • 3. Clique em "Escolher Arquivo" (em Inglês, "Choose file") e procure o arquivo dump.sql no diretório "fontes" (em Inglês, "sources") do pacote do template
    4. Clique no botão "Ir" (em Inglês, "Go") para começar a importar os dados de amostra do template

Instalação de Módulos

Os temas do Joomla são fornecidos com extensões adicionais: módulos, plugins e componentes que são utilizados para implementar várias funções e recursos. Todas as extensões adicionais podem ser instaladas em alguns cliques utilizando o pacote fornecido.

O arquivo do pacote está localizado no diretório 'fontes' do pacote do seu template e chamado de 'pkg_bundle.zip'. Pode ser facilmente instalado como qualquer outra extensão ou tema através do Painel de Controle do Joomla > Gerenciador de Extensões.

  1. Abra o painel de administração do Joomla. No menu superior selecione 'Extensões'. Na caixa suspensa selecione 'Gerenciador de Extensões';
  2. Certifique-se de que a seção 'Instalar' está selecionada na coluna da esquerda;
  3. Clique no botão 'Navegar/Escolher arquivo' e selecione o arquivo 'sources/pkg_bundle.zip';
  4. Clique no botão 'Fazer Upload e instalar' para instalar todos os módulos.

Se caso o seu tema contém algumas extensões que não têm sido incluídas no 'pkg_bundle.zip' você precisa instalar cada um separadamente. Isso poder ser feito exatamente como instalação do 'sources/pkg_bundle.zip' através da interface 'Gerenciador de Extensões'.

O nome dos arquivos das extensões geralmente começam com 'mod_', 'com_' ou 'plg_' então você irá reconhecê-los facilmente.

Extensões

A funcionalidade do site Joomla pode ser melhorada usando várias extensões do Joomla. Alguns deles são frequentemente utilizados em nossos templates. Essa seção irá lhe ensinar como mexer com extensões padrões.

Artigos de Última Hora

Img 11. Módulo Artigos de Última Hora (Avançado)

O módulo Artigos de Última Hora (Avançado) (em Inglês, "Articles Newsflash [Advanced]") é uma versão melhorada do módulo de Artigos de Última Hora do Joomla. Além das funções do módulo padrão, ele oferece opções de configuração extra e templates com base no módulo "Bootstrap".

Opções Adicionais

Número de colunas - permite mostrar artigos em várias colunas.

Link Padrão - lhe permite mostrar link padrão abaixo do conteúdo do módulo. Ele pode ser ligado a uma URL externa ou a algum item do menu do Joomla.

Img 12. Link Padrão

Pré-texto - permite mostrar texto padrão antes do conteúdo do módulo.

Data Publicada - mostra a data que o artigo foi publicado.

Criado por - mostra autor do artigo

Mostra imagens de introdução - permite mostrar imagem de introdução do artigo.

Alinhamento da imagem de introdução - permite alinhar as imagens de introdução

Image Swoop (em Português, "Movimento de Imagem")

Img 13. Módulo Image Swoop

O módulo "Image Swoop" é um slider sensível incrível desenvolvido como uma combinação de padrão do módulo "Articles Newsflash" do Joomla com o plug-in de fonte aberta gratuito em jQuery chamado "Camera Slideshow". Funciona perfeitamente no desktop e em aparelhos com iOS ou Android. Compatível com todos os navegadores modernos a começar pelo IE8.

O módulo "Image Swoop" possui várias opções de configuração, você pode acessá-las abrindo o módulo "Image Swoop" na seção Extensões > Gerenciador de Módulo (em Inglês, "Extensions > Module Manager") do seu painel de administração do Joomla. Todas as opções do plug-in em jQuery "Camera Slideshow" podem ser configuradas na área de configurações do módulo.

Img 14. Configurações do Image

Passe o cursor do mouse no título da opção para ver a descrição. Você também pode saber mais sobre as opções do plug-in em jQuery "Camera Slideshow" no site oficial

Articles Single (em Português, "Artigos Únicos")

O "Articles single" é um módulo que lhe permite colocar um único artigo na posição desejada.

Img 15. Módulo Articles Single

Img 16. Configurações do "Articles Single"

A única opção incomum no módulo é o Link Padrão. Ele lhe permite mostrar o link padrão abaixo do conteúdo do módulo. Ele pode ser ligado a uma URL externa ou a algum item do menu do Joomla.

CarouFredSel

O "CarouFredSel" é um módulo de carrossel de imagem desenvolvido como uma combinação do padrão do módulo "Articles Newsflash" do Joomla com o plug-in em jQuery "CarouFredSel".

Img 17. Módulo CarouFredSel

Todas as opções necessárias do carrossel estão disponíveis nas configurações do módulo.

  1. Direção do Carrossel
  2. Largura do Item
  3. Itens Visíveis
  4. Opções de Animação
  5. Navegação e controles

Img 18. Configurações do módulo CarouFredSel

O plug-in "CarouFredSel" tem muito mais opções de configuração. Se você quiser adicionar suas opções você pode editar o arquivo \modules\mod_caroufredsel\tmpl\default.php

O Komento é um componente de comentários para o Joomla. Ele lhe permite adicionar comentários aos artigos do Joomla.

Img 19. Componente do Komento

Img 20. Seção do componente do Komento

Ao usar configurações do componente você pode habilitar comentários para categorias específicas, adicionar captcha, usar perfis assim como nos artigos do Joomla, etc. Você pode saber mais sobre o Komento no site oficial

Nossos templates têm apenas a versão padrão do Komento incluso. Para ter acesso aos recursos adicionais e obter suporte completo à extensão Komento, recomendamos a compra do Komento Professional. A lista completa de recursos está disponível aqui.

Página da Galeria

Img 21. Imagens do Item Galeria

A página Galeria mostra posts da categoria 'Galeria' e suas sub-categorias. A página Galeria tem um filtro no canto superior direito para filtrar imagens por categorias.

Adicionando Imagens
Para adicionar mais imagens para a galeria, você precisa simplesmente adicionar novos artigos para as sub-categorias da 'Galeria'. Cada artigo deve ter imagem cheia e imagem de introdução enviada. Ambas as imagens são normalmente colocadas no diretório 'images/gallery'.

Img 22. Imagens do Item Galeria

A imagem de introdução é exibida dentro da galeria e das visualizações do item da galeria. A imagem cheia é exibida uma vez que a imagem de introdução for clicada.

Layout de Ajuste
A página da galeria pode ser ajustada como qualquer página da categoria Joomla. Abra a seção 'Menus > Menu Principal (em Inglês,"Menus > Menu Principal") do painel de administração do Joomla e edite o item do menu Galeria. Alterne para a aba Opções Avançadas (em Inglês, "Advanced Options") e você terá acesso as configurações da página Galeria.

Por exemplo, se você quiser alterar o número de colunas para a página Galeria, altere a opção Colunas (em Inglês, "Columns") da área de configurações "Opções do Layout do Blog (em Inglês, "Blog Layout Option").

Img 23. Número de colunas da página Galeria

Página de Contatos

A página de contatos exibe o formulário de contato, endereço e o Google Mapas. Usando as abas no topo você pode obter acesso a várias opções de contato.

Mapa do Google na página de contatos que é criada usando o plug-in do Google Mapas. Ele pode ser configurado através de Extensões > Gerenciador de Plug-in > Google Mapas (em Inglês, "Extensions > Plug-in Manager > Google Maps".)

Colocando vídeo no fundo (Background video)

Img 28. Vídeo de fundo na página do site

O vídeo de fundo pode ser adicionado ao template com a ajuda do plugin do jQuery chamado Vide. Verifique se o plugin está habilitado. Para descobrir isso, navegue para "Template manage -> Templates", então vá para o tema que você usa (ThemeXXXX arquivos e detalhes), abra o arquivo "index.php" e verifique se o plugin está sendo executado, como na imagem 29. Verifique, também, se o plugin está na pasta "js". No caso do plugin não estar lá, você pode baixá-lo usando o link acima e subir o arquivo para a pasta mencionada. Não esqueça de inserir o script para execução dele no arquivo index.php.

Img 29. Verificando a execução do plugin

Para definir o vídeo de fundo, navegue para "Template Manager -> Templates", clique no template que você usa (ThemeXXXX arquivos e detalhes). Na pasta "html -> mod_custom" crie um arquivo chamado ".php" com o seguinte conteúdo:

<?php
/**
* @package Joomla.Site
* @subpackage mod_custom
*
* @copyright Copyright (C) 2005 - 2012 Open Source Matters, Inc. All rights reserved.
* @license GNU General Public License version 2 or later; see LICENSE.txt
*/

defined('_JEXEC') or die;
?>
<div class="mod-custom mod-custom__<?php echo $moduleclass_sfx ?>" id="module_<?php echo $module->id; ?>">
<div class="video-container" data-vide-bg="<?php echo JURI::base( true ).'/media/video/[nome_do_arquivo]'; ?>"></div>
<div class="module-content">
<div class="module-content-inner">
<?php echo $module->content;?>
</div>
</div>
</div>

onde [nome_do_arquivo] é o nome do arquivo de vídeo, sem a extensão do arquivo.

Você também deve preparar o vídeo nos seguintes formatos: mp4, ogv, webm e um poster do vídeo em png, jpg e gif. O nome do arquivo deve ser o mesmo em todos os formatos e corresponder ao que você inserir no lugar de [nome_do_arquivo]. Todos os arquivos devem ser enviados para a pasta media\video na raíz do Joomla.

Img 30. Sobrescrevendo conteúdo do arquivo

No próximo passo, você precisa criar um módulo que apresenta o vídeo. Inclua um módulo HTML customizado com o seguinte conteúdo:

Img. 31. Configurações do módulo do vídeo de fundo

Em ambas as abas "Avançado" e no menu "Layout alternativo" escolha o arquivo de template [nome_do_arquivo] criado anteriormente.

Depois de você realizar todos os passos, você verá o módulo com o vídeo de fundo.

Usando o efeito de Parallax

O efeito de parallax pode ser adicionado ao template com a ajuda do plugin de jQuery Stellar.

Para adicionar o efeito na sua página, navegue para "Template manager -> Templates", clique no nome do template que você usa (ThemeXXXX arquivos e detalhes) e abra o arquivo "index.php". Selecione o bloco em que deseja adicionar o efeito de parallax, "folio-row" por exemplo, insira o atributo 

data-stellar-background-ratio="[valor]"
nesse elemento, como mostrado na Imagem 32:

Img 32. Usando o efeito parallax

onde [valor] é a proporção que será relativa a rolagem em velocidade natural, você pode procurar informações detalhadas na página de documentação do plugin.

Depois de você realizar todos os passos, você verá o efeito parallax no bloco específico.

Ambos marcação do HTML e estilos CSS usados em nossos temas são semânticamente corretos e válidos. Entretanto, alguns erros no validador do W3C ainda podem ocorrer. Nossa proposta é prover experiências ricas ao usuário com temas de alta qualidade e, para isso, temos que quebrar algumas regras.
