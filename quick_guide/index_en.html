Joomla! 3.x Quick Start Guide

Deploy Joomla! 3.x website in 3 simple steps

Step 1. Extract and Upload files

  • a.Open template folder
  • b.Extract files from the joomla/fullpackage.zip archive
  • c.Upload extracted directory to your hosting server.

Files can be extracted using any archive management applications like WinZip, Stuffit Expander or 7 Zip

To upload files to your server you can use File Manager from your hosting control panel or some third party solutions as FileZilla or Total Commander.

Note: you can extract fullpackage.zip directory to your server with joomla/unzip.php file. Upload it to your server and access using your browser address bar.

Step 2. create database

  • 1.Open your database management tool (usually phpMyAdmin) and click Databases tab
  • 2.Input new database name
  • 3.Click Create button
  • 4.You database will appear in the left column list

Step 3. Start Joomla! installation

Open web browser and type your Joomla! directory link http://path_to_your_joomla_directory

1

At the first installation step you need to input Joomla website name and description.

Into the right column fields please input website administrator details:

  • - email address
  • - username
  • - password
  • - password confirmation

Note: make sure to keep the administrator details safe

If Site Offline is enabled only website administrator will be able to access the site.

2

At the second step you need input database details:

  • - database type
  • - host name
  • - database username
  • - database user password
  • - database name
  • - table prefix

If you have difficulties with your database details please contact your hosting provider.

3

You are almost done! On the third step select sample data you want to install.

To make your template appear as at the preview page select sample_theme### option. (### is the number of your theme)

Before clicking Install button make sure your server matches Joomla! requirements.

All required options (left column) should be green with 'Yes' badge.

If all requirements are matched click Install button to proceed

congratulations!

Well done! Joomla is now installed.

Click Remove installation button to delete 'installation' directory.

Use 'Site' button to access your Joomla website or 'Administrator' button to get to the administration panel.
